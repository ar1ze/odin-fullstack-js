### Introduction

**Test Driven Development (TDD)** is a crucial concept in modern development. The main idea is simple: **you write automated tests *before* you write the code that is being tested.**

There are many JavaScript test-running systems (Mocha, Jasmine, Jest), but their basic syntax is very similar. This lesson focuses on **Jest** because of its excellent documentation and resources.

Remember, writing tests is less about the syntax and more about the TDD philosophy. Knowing **why** we write tests and **what** to test is more important than *how*.
