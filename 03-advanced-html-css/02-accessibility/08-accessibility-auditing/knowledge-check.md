**Question: What are some of the various accessibility features available in your browserâ€™s DevTools?**

**Answer:** Your DevTools allow you to view the **accessibility tree**, check **contrast ratios**, and view an element's **accessibility properties** (like its accessible name).

**Why:** These features are built-in to help you perform a "quick audit" and find accessibility issues. You can also use them to **emulate vision deficiencies** (like color blindness) or use the **Issues tab** to find problems.

***

**Question: Which third-party accessibility auditing tool is available in the Chrome DevTools by default?**

**Answer:** **Lighthouse** is available in Chrome DevTools by default.

**Why:** It's a built-in tool, often found in the "Auditing" or "Lighthouse" tab, that can run a comprehensive accessibility audit (along with checks for performance, best practices, and SEO).
